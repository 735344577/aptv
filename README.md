# APTV
## TG交流群
> [t.me/AptvPlayer](https://t.me/AptvPlayer)
## 介绍

一个用于播放 m3u8 文件的工具类App

## 区别
与现有的其他m3u8 播放器不同的是会实时获取视频预览图

## 已实现功能
- 添加远程m3u文件
- 播放m3u8文件
- 实时获取当前频道预览图
- 获取频道节目单（待优化）

## 接下来的计划
- iCloud 同步
- AppleTV 版本
